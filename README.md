# 🐟 Z&Z Peixaria

Site institucional simples e responsivo para a **Z&Z Peixaria**, desenvolvido como prática no curso de **Análise e Desenvolvimento de Sistemas (ADS)**.  
O projeto utiliza **HTML5, CSS3 e Bootstrap 5** para criar uma página moderna, com destaque para os produtos oferecidos e integração direta com o **WhatsApp** para pedidos.

---

## 🚀 Funcionalidades
- Página inicial com destaque para o filé de tilápia fresco  
- Cards de produtos com preços e botão de compra via WhatsApp  
- Seção de receitas para inspirar clientes  
- Botão flutuante de contato rápido pelo WhatsApp  
- Layout responsivo e otimizado para dispositivos móveis  

---

## 🛠️ Tecnologias utilizadas
- HTML5  
- CSS3  
- Bootstrap 5
