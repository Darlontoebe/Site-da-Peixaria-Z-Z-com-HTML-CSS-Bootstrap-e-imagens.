# Site da Peixaria Z&Z 
Landing page responsiva para peixaria local em Sobradinho-RS.  
Foco em filé de tilápia fresco, preços acessíveis e pedido rápido via WhatsApp.
PROJETO ainda em andamento, sera adicionado banco de dados e chatboot pra atendimento.
## Tecnologias usadas
- HTML5  
- CSS3  
- Bootstrap 5 (responsividade e cards bonitos)  
- Imagens customizadas (texturas e produtos reais)
## Acesse
[Acessar site](https://seu-link-netlify.netlify.app)  
(ou use GitHub Pages se ativar: https://darlontoebe.github.io/Site-da-Peixaria-Z-Z-com-HTML-CSS-Bootstrap-e-imagens/)
## Como rodar localmente
1. Clone o repo: `git clone https://github.com/Darlon toebe/Site-da-Peixaria-Z-Z-com-HTML-CSS-Bootstrap-e-imagens.git`  
2. Abra `index.html` no navegador.

Projeto feito durante o curso de ADS para prática de front-end e deploy.
